Overview

This tutorial presents a unified view of Scientific Machine Learning (SciML) for control-oriented decision making. We frame modeling, optimization, and control as differentiable programs that combine physics-based structure, data-driven learning, constraints, and gradient-based optimization.

Learning to Model

Constrained system identification, neural ODEs, structured learning, and differentiable simulators.

Learning to Optimize

Parametric optimization, differentiable projection, and learning policies for constrained programs.

Learning to Control

Differentiable Predictive Control, approximate explicit MPC, and neural operator control for PDEs.

Setup

Participants can run most examples directly in Google Colab. For local execution, clone the repository and install the Python dependencies below. 

git clone https://github.com/YOUR-ORG/L4DC-SciML-Tutorial.git
cd L4DC-SciML-Tutorial
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
